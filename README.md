# Лабораторное занятие №6 — самостоятельная часть (обычная сложность)

## Установка

1. Создать виртуальное окружение: py -m venv venv
2. Активировать виртуальное окружение: .\venv\Scripts\activate
3. Установка зависемости: pip install -r requirements.txt

## Запуск модулей

1. Конвертация температур (F → K): python src/convert_temp.py
2. Построение графиков: python src/plots.py
3. Математические задачи: python src/math_tasks.py
